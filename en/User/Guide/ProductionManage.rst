Manage media in *Phraseanet Production*
=======================================

.. topic:: The essential
	
	Media management is in *Phraseanet Production*.
	This essential interface regroups all the functions allowing to manage,
	cast and publish assets.
	A general introduction of the *Production* interface is in the previous
	section of this guide.
	In the following pages, only the advanced features are described. They
	concern users whom have missions to manage and animate media assets.
